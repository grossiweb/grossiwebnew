# Sample GraphQL Queries for Testing

## Test Basic Connection
```graphql
query {
  posts {
    nodes {
      title
      content
    }
  }
}
```

## Test Homepage Data
```graphql
query GetHomepage {
  page(id: "home", idType: URI) {
    id
    title
    content
    customFields {
      heroTitle
      heroSubtitle
      heroVideo
      heroWords {
        word
      }
    }
  }
}
```

## Test Services

query GetServices {

## Test Menu
```graphql
query GetMenu($location: MenuLocationEnum!) {
  menuItems(where: { location: $location }) {
    nodes {
      id
      label
      url
      path
      parentId
      cssClasses
      description
      target
      title
    }
  }
}
```
